## 详细说明

### 1. 顶部导航栏（固定，sticky）
```
[←] Details                    [Active]
```
- 左侧：返回按钮 + 标题 "Details"
- 右侧：状态徽章（Active/Exited/Error/Abnormal）

### 2. 照片区域（Card 1）
```
┌─────────────────────────┐
│                         │
│     车辆照片 (16:9)      │
│                         │
└─────────────────────────┘
```
- **有照片**：显示车辆照片
- **无照片**：显示占位符（车辆图标 + "No photo"）
- **加载失败**：显示错误提示（图片图标 + "Photo unavailable"）

### 3. 车牌号区域（Card 2）
```
Plate
─────────────────
ABC1234      [✏️]
Original: XYZ5678
```
- 大号等宽字体显示车牌号
- 编辑按钮（仅 active 且未修改过）
- 原始车牌号（如已修改）

### 4. 时间信息区域（Card 3）
```
📅 Entry      Jan 15, 2:30 PM
📅 Exit       Jan 15, 5:45 PM
─────────────────────────────
🕐 Time           3h 15m
```
- Entry：入场时间
- Exit：出场时间（如有）
- Time：停车时长（大号字体）

### 5. QR 码区域（Card 4，仅 active 状态）
```
     QR Code
  ┌─────────┐
  │         │
  │ QR Code │
  │         │
  └─────────┘
```

### 6. 设备信息
```
📱 Device: abc123def456...
```
- 小字显示设备 ID（截取前 20 字符）

### 7. 操作按钮（仅 active 状态）
```
┌─────────────────────────┐
│  [🚪 Confirm Exit]      │
└─────────────────────────┘
```
- 橙色按钮，点击后弹出确认对话框

## 状态说明

### Active 状态显示：
- ✅ 照片区域
- ✅ 车牌号（可编辑）
- ✅ 时间信息
- ✅ QR 码
- ✅ 设备信息
- ✅ 退出按钮

### Exited 状态显示：
- ✅ 照片区域
- ✅ 车牌号（不可编辑）
- ✅ 时间信息（包含 Exit）
- ❌ QR 码
- ✅ 设备信息
- ❌ 退出按钮

### 编辑车牌模式：
```
Plate
─────────────────
[ABC1234] [✓] [✗]
```
- 输入框 + 确认按钮 + 取消按钮

## 退出确认对话框

```
┌─────────────────────────┐
│  Confirm Exit           │
│                         │
│  Exit plate ABC1234?    │
│  3h 15m                 │
│                         │
│  [Cancel] [Confirm Exit]│
└─────────────────────────┘
```

## 响应式设计

- 最大宽度：`max-w-lg`（约 512px）
- 内边距：`px-4 py-6`
- 卡片间距：`space-y-4`
- 底部安全区域：`pb-24`（为底部导航栏留空间）
